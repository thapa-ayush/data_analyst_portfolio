# Professional Spacing & Sizing Guidelines

## UNIFIED SPACING SYSTEM

### Desktop (1440px+)

- **Section Padding:** 5rem (top/bottom), 2rem (left/right)
- **Section Gap:** 3rem between sections
- **Container Max-Width:** 1200px
- **Grid Gap:** 2.5rem (projects, certificates)
- **Card Padding:** 1.5-2rem
- **Button Padding:** 1.1rem 2.5rem
- **Border Radius:** 12px (containers), 8px (cards)

### Laptop (1024px - 1440px)

- **Section Padding:** 4rem (top/bottom), 2rem (left/right)
- **Section Gap:** 2.5rem
- **Container Max-Width:** 1000px
- **Grid Gap:** 2rem
- **Card Padding:** 1.25rem
- **Button Padding:** 0.95rem 2rem
- **Column Collapse:** 3 columns → 2 columns

### Tablet (768px - 1024px)

- **Section Padding:** 3rem (top/bottom), 1.5rem (left/right)
- **Section Gap:** 2rem
- **Container Max-Width:** 100% (with margins)
- **Grid Gap:** 1.5rem
- **Card Padding:** 1rem
- **Button Padding:** 0.9rem 1.5rem
- **Column Collapse:** 2 columns → 1 column

### Mobile (480px - 768px)

- **Section Padding:** 2.5rem (top/bottom), 1rem (left/right)
- **Section Gap:** 1.5rem
- **Container Max-Width:** 100%
- **Grid Gap:** 1rem
- **Card Padding:** 0.75rem
- **Button Padding:** 0.85rem 1.25rem
- **Column Collapse:** Ensure single column

### Small Mobile (320px - 480px)

- **Section Padding:** 2rem (top/bottom), 0.75rem (left/right)
- **Section Gap:** 1.25rem
- **Container Max-Width:** 100%
- **Grid Gap:** 0.75rem
- **Card Padding:** 0.75rem
- **Button Padding:** 0.75rem 1rem
- **Font-Size Reduction:** -10% from tablet

## HEADING HIERARCHY

- **H1:** 3.5-4.5rem (desktop), 2rem (mobile)
- **H2:** 2.5-3rem (desktop), 1.5rem (mobile)
- **H3:** 1.5-2rem (desktop), 1.2rem (mobile)
- **Label:** 0.8-0.9rem (consistent across devices)

## BUTTON STANDARDS

All buttons should be:

- Minimum height: 44px (touch target)
- Consistent styling across all pages
- Same gradient/colors
- Responsive width on mobile (100%)

## GRID SPECIFICATIONS

**Desktop:** 3 columns
**Laptop:** 2-3 columns (depending on content)
**Tablet:** 2 columns
**Mobile:** 1 column
**Gap:** Adjust per breakpoint
